# Validator

This is validator of a bridge between Ethereum and Substrate.

# Status

POC. Active development.

# Building

Install Rust:

```bash
curl https://sh.rustup.rs -sSf | sh
```

Build:

```bash
cargo build
```

Prepare `.env` file with default environment variables:

```bash
cp .env.example .env
```

# Run

```bash
cargo run
```
